# Alba Sona Makeup - Portfolio Profesional 💄

Proyecto web desarrollado para mostrar el trabajo y los servicios de maquillaje profesional de Alba Sona. Diseñado con un enfoque en el rendimiento extremo, la elegancia visual y la experiencia de usuario (UX).

## 🚀 Tecnologías Utilizadas

- **[Astro](https://astro.build/):** Framework principal utilizado por su arquitectura de islas y su renderizado estático superrápido.
- **HTML5 & CSS3:** Estilos nativos encapsulados por componentes y uso de variables CSS globales para mantener la coherencia de la identidad de marca.
- **Astro Assets:** Sistema de optimización automática para imágenes de alta resolución en formato WebP.
- **ViewTransitions (ClientRouter):** Implementación de navegación nativa tipo SPA (Single Page Application) para transiciones de página fluidas y sin parpadeos.
- **Vercel:** Plataforma de despliegue y alojamiento continuo.

## ✨ Características Principales

- **Rendimiento Perfecto (Lighthouse):** Carga ultrarrápida gracias a la eliminación de JavaScript innecesario en el cliente.
- **Diseño Responsivo 'Mobile-First':** Adaptación perfecta a cualquier dispositivo, con ajustes específicos de usabilidad en menús y galerías para móviles.
- **SEO Optimizado:** Estructura semántica, metaetiquetas dinámicas y tiempos de carga reducidos.
- **Imágenes Vectoriales (SVG):** Uso de gráficos vectoriales e integraciones CSS (máscaras/filtros) para logotipos en alta definición sin pérdida de calidad.
